# Core Feature Set (UK Consumer Retirement App)

Date: 2026-04-29
Positioning: Consumer-first, UK-focused, simple UX by default with advanced capabilities available on demand.

## Tier A: Core (v1 launch)
1. Guided onboarding (age, target retirement age, current assets, contributions, spending target)
2. Deterministic projection engine (yearly cash flow + portfolio balance with inflation)
3. Retirement readiness outputs (surplus/shortfall, years-funded, suggested safe spending)
4. Scenario builder (duplicate and compare plans)
5. UK basics (State Pension input assumptions, ISA/SIPP account handling, baseline tax estimate)
6. Decision dashboard (top prioritized actions to improve plan success)

## Tier B: Advanced (optional modules)
1. Monte Carlo simulation and probability of success
2. Sequence-of-returns risk visualization
3. Decumulation/withdrawal strategy modeling
4. Advanced UK tax treatment in retirement drawdown
5. Stress testing (inflation shocks, low return decade, longevity variance)
6. Goal prioritization/trade-off tooling

## Tier C: Later roadmap
1. Advisor mode and multi-client workflows
2. Estate/inheritance deep planning
3. Open banking / aggregation integrations
4. AI copilot for natural-language planning workflows

## Definition of advanced features
Advanced features generally require one or more of:
- stochastic simulation,
- multi-rule tax/withdrawal optimization,
- cross-goal trade-off solving,
- nonlinear stress scenario analysis.
